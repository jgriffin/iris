DetectionLayer · skeleton-rendering reference

The synthetic body-pose fixture from DetectionLayer.previewStore() rendered as the SwiftUI Canvas draws it: every Skeleton.humanBodyPose edge whose both endpoints resolve to a present joint is stroked, then each joint gets a filled dot. The skeleton topology is carried on the Detection itself (detection.skeleton) — the overlay holds no joint knowledge. Coordinates apply the same Y-flip the converter does (y_screen = (1 − y_norm) · height, Vision bottom-left origin → screen top-left), so the head (high normalized y) is at the top. The label anchors to the bounding-box top-left, exactly like the box/quad paths. Open in a browser; no build step.
