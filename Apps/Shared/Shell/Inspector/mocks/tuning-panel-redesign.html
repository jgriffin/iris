Iris — Tuning / inspector panel redesign

Static mock of the redesigned right-inspector (macOS, ~300pt docked). No “Tuning” title, no DETECTOR / DISPLAY sub-headers. Prominent detector name on top, ONE global Min confidence, then a dense one-line-per-class list. The reset icon sits immediately left of the value and appears ONLY on overridden rows, so the value column stays aligned. Three per-class-row interaction variants + a full COCO roster column.
